1. Project Overview
SkillBridge is a full-featured Learning Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The platform supports three distinct user roles — Student, Instructor, and Admin — each with dedicated dashboards and role-based permissions.
The system enables students to browse and enroll in courses with progress tracking, instructors to create and manage educational content with lessons, and administrators to oversee the entire platform with analytics and user management.
Key Features
•  Role-Based Access Control (Admin, Instructor, Student) with JWT authentication
•  Full CRUD operations for courses and lessons with category filtering
•  Student enrollment system with progress tracking and personal dashboard
•  Instructor course management with lesson upload capability
•  Admin analytics dashboard with platform-wide statistics
•  Search and category-based filtering on the course catalog
•  Protected routes with role-based middleware authorization
•  Responsive, professional UI design with custom CSS and Bootstrap 5
•  Password hashing with Bcrypt and environment variable configuration
